README语法简介

发表于 2017-10-11

(1)标题

我们使用#来确定标题,这里相当于HTML中的H1-H6,#越少标题字体就越大,当然要从一个开始算起,最大是六个

(2)图片导入

导入图片是用![图片名称](图片路径)

(3)网址或者链接导入

链接或者网址导入[网址名称](网址路径)

(4)强调

我们可以使用下面的方式给我们的文本添加强调的效果

*强调* 或者 _强调_ (示例：斜体)
**加重强调** 或者 __加重强调__ (示例：粗体)
***特别强调*** 或者 ___特别强调___ (示例：粗斜体)

(5)代码引入

如果我们想在文章中添加代码，我们有两种方式

第一种方式是使用反引号(esc键下面的按钮)将代码包裹起来

第二种方式则是使用制表符或者至少4个空格进行缩进的行

(6)换行

如果我们想把一行文本进行换行，我们可以在需要换行的地方输入至少两个空格，然后回车即可，注意，如果不回车，是没有效果的

(7)引用资料

如果我们在文章中引用了资料，那么我们可以通过一个右尖括号”>”来表示这是一段引用内容。我们可以在开头加一个，也可以在每一行的前面都加一个。我们还可以在引用里面嵌套其他的引用

(8)分割线

如果我们想用分割线对内容进行分割，我们可以在单独一行里输入3个或以上的短横线、星号或者下划线实现。短横线和星号之间可以输入任意空格

(9)列表标记

我们使用列表的时候,我们可以是用三种方式:

第一种:阿拉伯数字

第二种:使用-,每一行前面使用一个-,在页面实现的效果就是圆点

第三种:使用*,每一行前面使用一个*,在页面实现的效果也是圆点
